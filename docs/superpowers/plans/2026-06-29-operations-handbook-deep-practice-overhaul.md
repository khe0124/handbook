# Operations Handbook Deep Practice Overhaul Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Upgrade the operations handbook from a strong conceptual/playbook guide into a practical incident-training handbook with realistic commands, outputs, logs, metrics, timelines, and interpretation drills.

**Architecture:** Keep the existing generated handbook model, but extend each operations document with a new `practiceLabs` section that renders concrete “observe → interpret → act” exercises. Each lab should include a scenario, command/query, normal sample, abnormal sample, diagnosis, mitigation, and permanent fix. Tests must verify that representative AWS, Azure, Kubernetes, Terraform, PromQL, log, and postmortem examples exist and that generic training prose does not reappear.

**Tech Stack:** Node.js ESM generator, static HTML output, generated TypeScript document modules, Node built-in test runner.

- [ ] **Step 2: Add lab renderer**

```js
const renderPracticeLabs = (labs) =>
  labs
    .map(
      (lab, index) => `        <div class="practice-lab">
          <span class="sc-label">PRACTICE LAB ${String(index + 1).padStart(2, "0")} · ${escapeHtml(lab.title)}</span>
          <p>${escapeHtml(lab.scenario)}</p>
          <table>
            <tr><th>Command / Query</th><th>목적</th></tr>
            <tr><td>${renderCodeBlock(lab.command)}</td><td>${escapeHtml(lab.purpose)}</td></tr>
          </table>
          <table>
            <tr><th>정상 출력</th><th>비정상 출력</th><th>판단 훈련</th></tr>
            <tr><td>${renderCodeBlock(lab.normalOutput)}</td><td>${renderCodeBlock(lab.abnormalOutput)}</td><td>${escapeHtml(lab.interpretation)}</td></tr>
          </table>
          <table>
            <tr><th>즉시 완화</th><th>영구 수정</th><th>검증 기준</th></tr>
            <tr><td>${escapeHtml(lab.mitigation)}</td><td>${escapeHtml(lab.permanentFix)}</td><td>${escapeHtml(lab.verification)}</td></tr>
```
