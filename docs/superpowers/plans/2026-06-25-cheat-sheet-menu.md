# Cheat Sheet Menu Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Add a new `Cheat Sheet` top-level menu with eight compact reference pages.

**Architecture:** Follow the existing generated-document flow. Add `CHEAT_SHEETS` to `src/handbook/catalog.mjs`, create one source HTML file per cheat sheet under `public/handbook`, run the generator to create `src/handbook/documents/*.ts` and `src/handbook/documentLoaders.ts`, and add practical example coverage so current tests continue to enforce complete handbook metadata.

**Tech Stack:** React 19, TypeScript, Vite, Node test runner, static handbook HTML extraction.
