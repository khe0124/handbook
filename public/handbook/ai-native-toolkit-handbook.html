KIT-01

템플릿 사용 원칙

원칙설명
작업 전 작성완료 후 회고가 아니라 작업 시작 전에 제약과 검증을 고정한다
AI와 사람 분리AI가 제안한 내용과 사람이 승인한 근거를 구분한다
재현 가능성다른 사람이 같은 입력으로 같은 검증을 실행할 수 있어야 한다
포트폴리오 연결실무 산출물을 면접 증거로 재사용할 수 있게 보존한다

  - "npm test -- invite"
  - "npm run build"

  - "만료/이미 사용/권한 없음 상태가 분리된다"

  - "feature flag invite_error_v2 off"
