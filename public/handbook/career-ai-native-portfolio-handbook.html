PORT-04

30초/90초 답변 카드

길이구조
30초문제 → 내가 맡은 범위 → 핵심 판단 → 결과 지표
90초문제 → 대안 → 구현 → 평가/보안/운영 → 실패와 개선 → 배운 점
좋은 답변

문서 검색 기능에서 30개 golden query를 만들고 recall@5와 citation precision을 릴리스 기준으로 뒀습니다. prompt 변경 때마다 baseline/candidate report를 남겼고, prompt injection fixture를 회귀 테스트에 포함했습니다.
