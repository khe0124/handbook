RECOMMENDED PATH
1. 기본기와 제품 구현

개발 핸드북과 디자인 실무로 사용자 흐름, UI/API/DB 계약, 테스트 기준을 잡습니다.

2. LLM 제품화

LLM 문서로 프롬프트, RAG, 평가, 보안, 운영 구조를 설계합니다.

3. AI Native 훈련

역량 매트릭스와 실습 랩으로 fixture, pass criteria, evidence pack을 통과합니다.

4. 운영과 증거화

인프라·운영, AX, 커리어 문서로 runbook, episode, 포트폴리오 증거를 남깁니다.
